  ```
<=> sudo apt-install git
<=> git clone https://github.com/dimas7891/M3_DS.git
<=> cd M3_DS
<=> chmod +x install.sh
<=> chmod +x M3_DS
<=> ./install.sh
<=> ./M3_DS
 
  ```
